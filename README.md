# xuggler

A Clojure wrapper for Xuggler (https://github.com/xuggle/xuggle-xuggler)

## Usage

```clojure
;clojar coming soon!

(:use [xuggler.core :only [convert]])
(convert "input.flv" "output.mp4")
```
